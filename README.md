# 🛡️ DeepSentinel
### Real-Time Social Media Crisis Detection System

A deep learning system that detects social media crises in real-time 
using a three-branch neural architecture.

## 🚀 Features
- Detects 5 crisis types: Misinformation, Mental Health, Cyberbullying, Political Unrest, Normal
- BERT + Bi-LSTM + GNN Fusion architecture
- Real-time Streamlit dashboard
- Multi-platform monitoring (Twitter/X, Reddit, NewsAPI)

## 🛠️ Tech Stack
- Python, PyTorch, HuggingFace Transformers
- Streamlit, Plotly
- torch-geometric (GNN)

## ▶️ Run
pip install -r requirements.txt
streamlit run 05_dashboard.py
